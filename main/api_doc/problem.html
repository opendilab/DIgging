digging.problem¶

ProblemHandler¶

class digging.problem.ProblemHandler(search_space: BaseSpace)[source]¶

This class is used to provide a uniformed interface for the digger to interact with. It can store all the searched samples and scores together with the best ones. Users can provide a convert function to convert sample into data array if the sample is not in a standard array type.

Parameters:

  • search_space (BaseSpace) – Search space of handler.

  • sample_converter (Callable) – Convert func to convert samples to data, defaults to None

DiscreteSpace¶

class digging.problem.DiscreteSpace(shape: ~typing.Optional[~typing.List] = None, dtype: ~typing.Type = <class 'numpy.int64'>, random_state: ~typing.Optional[~typing.Any] = None)[source]¶

Discrete search space. It is configured by a input value or a list. The element of input at each position determinates the discrete space for each dim.

Parameters:

  • shape (List) – List of space shape, defaults to None.

  • dtype (Type) – Data type of samples, defaults to np.int64.

  • random_state (Any) – Random state of sampler, defaults to None.

Examples:

>>> space = DiscreteSpace((2, 3, 4))
>>> space.shape
array([3, ])
>>> x = space.sample()
>>> x
array([1, 1, 0])

ContinuousSpace¶

class digging.problem.ContinuousSpace(shape: ~typing.List, low: ~typing.Optional[float] = None, high: ~typing.Optional[float] = None, dtype: ~typing.Type = <class 'numpy.float32'>, random_state: ~typing.Optional[~typing.Any] = None)[source]¶

Continuous search space. The space is defined by its shape and lower/higher bounds, with continuous value. Users can set the lower and higher boundary of each dimension of the space, otherwise the value will be infinite at these dimensions. Each dimension can have different boundary types and values. If so, the shape of boundary argument must be the same as space’s shape.

Parameters:

  • shape (List) – Shape of the space.

  • low (float) – The lower bound of space, defaults to None.

  • high (float) – The higher bound of space, defaults to None.

  • dtype (Type) – Data type of space, defaults to np.float32.

  • random_state (Any) – Random space of sampler, defaults to None.

Example:

>>> space = ContinuousSpace(shape=(2, 3), low=0, high=((5, 5, 5), (8, 8, 8)))
>>> space.shape
array([2, 3])
>>> space.sample()
array([[2.7791994, 4.216622 , 3.0280395],
   [4.2073464, 7.35141  , 2.5952444]], dtype=float32)
